팀 업무를 주차별로 관리하는 앱이에요.
Google Sheets와 연동해서 팀원 모두가 볼 수 있어요.

1단계 — Google Sheets API 키

Google Cloud Console에서 API 키 발급 → Sheets API 활성화

2단계 — 스프레드시트 ID

구글 시트 URL에서 /d/ 와 /edit 사이 부분을 복사하세요

3단계 — 프로젝트 이름 (선택)

💡 시트 구조: 시트가 없으면 자동 생성됩니다. 시트는 공개 또는 API 키 접근 가능으로 설정하세요.
